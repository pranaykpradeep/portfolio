Buggy Project - Detailed Review

Executive Summary

The Feugo 15 project is an initiative to design and build a buggy powered by a 150cc Hero CBZ petrol engine. This project serves as both a technical challenge and a platform to demonstrate our engineering skills.

Introduction

The F15 buggy bridges theoretical learning with hands-on application by combining principles of mechanical design, electronics, and control systems.

Key Features Include:

  • Engine: 150cc Hero CBZ petrol engine.
  • Chassis: Open design using rectangular steel pipes with integrated suspension and measurement provisions.
  • Electronics: Arduino/ESP boards with sensor integration and potential LoRa/GSM communication.
  • Control: Mobile app management for engine control, key activation, and emergency safety functions.
  • Events: Hosting workshops, technical talks, webinars, and driving events.

Technical Approach

The project involves multiple technical aspects:

  • Mechanical Design: Lightweight yet durable chassis using rectangular steel pipes.
  • Electrical & Electronics: Real-time control using Arduino/ESP boards and sensor integration.
  • Control Systems: Mobile app for key activation, engine start/stop, and emergency kill switch.

Timeline and Milestones

We have set specific milestones to ensure timely progress:

  • Design Phase: 1 week - Finalize sketches and CAD models.
  • Component Sourcing: 1 week - Confirm dimensions, secure vendors, select materials.
  • Fabrication: 1-2 weeks - Chassis construction, suspension installation, engine mounting.
  • Electronics Integration: 1 week - Develop firmware, integrate sensors, implement controls.
  • Testing: 1 week - Field testing and adjustments.
